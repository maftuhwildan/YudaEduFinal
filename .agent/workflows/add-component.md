# Add ShadcnUI Component

1. Add the component using the shadcn CLI:
```bash
npx shadcn add <component-name>
```

Example components: button, card, dialog, input, select, tabs, table, form

2. The component will be installed in `components/ui/`

3. Import in your code:
```tsx
import { Button } from "@/components/ui/button"
```

## Common Components

- `button` - Buttons with variants
- `card` - Card container
- `dialog` - Modal dialogs
- `input` - Form inputs
- `select` - Dropdown select
- `tabs` - Tabbed interface
- `table` - Data tables
- `form` - Form with validation
- `scroll-area` - Scrollable container
